# Trade Alert Widget

## Introduction

This plugin creates graphs with live trading data by using shortcodes.

## Code examples

>  Use [trade-alert]  to display the two primary graphs with live trading information use [trade-alert alert='primary'] to only show the first graph and  [trade-alert alert='secondary'] to display the second graph.  Additionally [trade-history] will display a chart with historical data.

## Installation

> Upload the plugin as a zipped file via plugins-> add new - > upload plugin and after uploading activate the plugin by clicking activate on the plugin menu page.

The shortcodes listed above will now work in any text area including within widget areas
